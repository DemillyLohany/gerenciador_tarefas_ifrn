# SeuChecklist 

## Integrantes: Camila Thaís Silva Medeiros, Demilly Lohany Gonçalves de Medeiros e Filipe Silva Souza Marcelino

Descrição: O sistema consiste em uma aplicação web para gerenciamento de tarefas e produtividade, desenvolvida com o objetivo de auxiliar o usuário na organização de suas atividades diárias. A plataforma permite o cadastro e acompanhamento de tarefas, controle de status, priorização por prazo e uso de técnicas de foco, como o método Pomodoro, visando melhorar a produtividade e o gerenciamento de tempo.

Frontend: NextJS

Backend: FastAPI (Python)

Banco de Dados: SqlModel

## Funcionalidades Principais
 - Cadastro e autenticação de usuários

 - Gerenciamento completo de tarefas (CRUD)

 - Controle automático de status (pendente, concluída, atrasada)

 - Priorização de tarefas com base na data prevista

 - Temporizador Pomodoro para foco em atividades

 - Relatório semanal de desempenho do usuário
