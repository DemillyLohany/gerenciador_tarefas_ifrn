# Nome do Projeto: SeuChecklist - Gerenciador de tarefas

Componentes: Camila Thaís Silva Medeiros, Demilly Lohany Gonçalves de Medeiros e Filipe Silva Souza Marcelino

Descrição: O projeto desenvolvido funcionará como um gerenciador de tarefas, cujo objetivo está na organização e manutenção sustentável de afazeres que precisam ser concluídos. Ele irá possibilitar visualização, cadastro, edição e, finalmente, a conclusão, mas somente comprovada, das tarefas de responsabilidade de seus usuários.
